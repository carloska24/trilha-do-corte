# Database Designer

## Trigger

- criar banco de dados
- modelar banco
- database
- schema
- tabelas
- relacionamentos

## Inputs

### Required

- `entidades_do_dominio`

### Optional

- `volume_de_dados`
- `tipo_de_banco`
- `consultas_criticas`

## Steps

1. **database_architect**

   - Task: Definir tipo de banco e estratégia de modelagem.

2. **data_modeler**

   - Task: Criar entidades, relacionamentos e normalização.

3. **performance_engineer**

   - Task: Definir índices e otimizações.

4. **migration_engineer**

   - Task: Planejar migrações e versionamento.

5. **consistency_guard**
   - Task: Garantir integridade referencial e constraints.

## Output Deliverables

- Modelo de dados (texto)
- Tabelas e relacionamentos
- Índices recomendados
- Estratégia de migração
- Boas práticas de uso
