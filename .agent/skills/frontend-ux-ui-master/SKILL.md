# Frontend UX/UI Master

## Trigger

- criar frontend
- criar interface
- criar UI
- criar UX
- frontend profissional
- dashboard
- landing page
- aplicação web

## Steps

1. **ux_designer**

   - Task: Definir fluxo do usuário, personas, objetivos e hierarquia da informação.

2. **ui_designer**

   - Task: Criar identidade visual, layout, grid, tipografia, cores, espaçamentos e componentes.

3. **frontend_engineer**

   - Task: Implementar a interface usando Next.js, React, TypeScript, Tailwind e shadcn/ui.

4. **motion_designer**

   - Task: Adicionar animações suaves, microinterações e feedback visual usando Framer Motion.

5. **accessibility_engineer**

   - Task: Garantir acessibilidade WCAG, navegação por teclado e leitores de tela.

6. **performance_engineer**
   - Task: Otimizar performance, carregamento, imagens e estrutura de componentes.

## Output Deliverables

- Estrutura de pastas
- Componentes React reutilizáveis
- Layout responsivo
- Design system
- Código limpo e comentado
- Sugestões de melhorias UX
