# Product Designer

## Trigger

- definir produto
- criar mvp
- planejar funcionalidades
- roadmap
- ideia de produto
- saas
- plataforma
- app

## Inputs

### Required

- `descricao_do_produto`

### Optional

- `publico_alvo`
- `problema_principal`
- `modelo_de_negocio`
- `concorrentes`

## Steps

1. **product_manager**

   - Task: Identificar problema central e proposta de valor.

2. **product_designer**

   - Task: Definir jornada do usuário e fluxos principais.

3. **business_analyst**

   - Task: Definir modelo de negócio e métricas de sucesso.

4. **tech_lead**

   - Task: Validar viabilidade técnica e dependências.

5. **prioritization_specialist**
   - Task: Priorizar funcionalidades usando impacto vs esforço.

## Output Deliverables

- Descrição clara do MVP
- Lista de funcionalidades (Must / Should / Later)
- Fluxos principais do usuário
- Roadmap inicial
- Critérios de sucesso
