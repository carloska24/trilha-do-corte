# Security Audit Agent

## Trigger

- auditar seguranca
- security
- vulnerabilidades
- proteger sistema
- seguranca api
- auditoria

## Inputs

### Required

- `arquitetura_do_sistema`

### Optional

- `codigo`
- `tipo_de_autenticacao`
- `ambiente`

## Steps

1. **application_security_engineer**

   - Task: Avaliar superfícies de ataque da aplicação.

2. **auth_security_specialist**

   - Task: Auditar autenticação, autorização e permissões.

3. **data_protection_engineer**

   - Task: Verificar proteção de dados sensíveis.

4. **input_validation_auditor**

   - Task: Analisar validações e prevenção de ataques.

5. **security_best_practices_guard**
   - Task: Aplicar boas práticas e recomendações.

## Output Deliverables

- Lista de vulnerabilidades
- Riscos priorizados
- Recomendações de correção
- Checklist de segurança
- Boas práticas aplicáveis
