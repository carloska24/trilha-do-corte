# System Architecture Designer

## Trigger

- criar sistema
- criar aplicação
- arquitetura
- planejar sistema
- novo projeto
- saas
- app
- plataforma

## Inputs

### Required

- `tipo_de_produto`

### Optional

- `publico_alvo`
- `escala_esperada`
- `orcamento`
- `requisitos_legais`

## Steps

1. **software_architect**

   - Task: Definir modelo arquitetural (monolito, microserviços, serverless).

2. **tech_lead**

   - Task: Selecionar stack tecnológica adequada.

3. **cloud_architect**

   - Task: Definir infraestrutura e estratégia de deploy.

4. **security_architect**

   - Task: Identificar riscos de segurança e mitigação.

5. **cost_analyst**
   - Task: Avaliar custo inicial e crescimento.

## Output Deliverables

- Diagrama arquitetural (texto)
- Stack recomendada
- Fluxo de dados
- Decisões técnicas justificadas
- Riscos e mitigação
