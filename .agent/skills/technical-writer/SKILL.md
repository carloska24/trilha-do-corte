# Technical Writer

## Trigger

- documentar projeto
- criar readme
- documentacao tecnica
- docs
- documentar api

## Inputs

### Required

- `descricao_do_sistema`

### Optional

- `stack`
- `arquitetura`
- `publico`

## Steps

1. **technical_writer**

   - Task: Criar README claro e estruturado.

2. **architectural_writer**

   - Task: Documentar arquitetura e decisões técnicas.

3. **api_writer**

   - Task: Documentar endpoints, contratos e exemplos.

4. **onboarding_specialist**

   - Task: Criar guia de setup e contribuição.

5. **maintenance_guard**
   - Task: Garantir clareza e atualização contínua.

## Output Deliverables

- README profissional
- Documentação de arquitetura
- Docs de API
- Guia de setup
- Guia de deploy
- Boas práticas
