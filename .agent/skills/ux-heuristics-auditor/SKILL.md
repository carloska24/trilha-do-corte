# UX Heuristics Auditor

## Trigger

- avaliar ux
- melhorar usabilidade
- auditoria ux
- fluxo do usuario
- experiencia do usuario
- ux

## Inputs

### Required

- `fluxos_principais`

### Optional

- `tipo_de_usuario`
- `contexto_de_uso`
- `plataforma`

## Steps

1. **ux_researcher**

   - Task: Avaliar visibilidade de status do sistema.

2. **ux_designer**

   - Task: Analisar consistência, padrões e feedback visual.

3. **interaction_designer**

   - Task: Identificar fricções, cliques desnecessários e confusão.

4. **accessibility_specialist**

   - Task: Verificar acessibilidade básica e inclusão.

5. **heuristics_auditor**
   - Task: Aplicar heurísticas de Nielsen e padrões modernos.

## Output Deliverables

- Lista de problemas de usabilidade
- Sugestões práticas de melhoria
- Fluxos otimizados
- Boas práticas recomendadas
- Checklist UX antes do desenvolvimento
